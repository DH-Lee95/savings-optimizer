# PRD: 최적 저축 포트폴리오 서비스

## 목표
사용자의 목돈, 월 저축액, 사용 시점, 유동성 필요 여부를 바탕으로 예금, 적금, 파킹통장 조합의 세후 예상 이자를 계산하고 이해하기 쉬운 배분안을 제공한다.

## 사용자
- 월급을 받기 시작한 20대, 30대 직장인
- 목돈 500만 원에서 5,000만 원 규모를 안정적으로 굴리고 싶은 사용자
- 전세, 결혼, 자동차, 여행, 유학 등 목적자금을 준비하는 사용자
- 예금, 적금, 파킹통장 차이와 우대금리 조건이 어려운 금융 초보자

## 핵심 기능
1. 사용자 입력 폼: 월 저축 가능액, 현재 목돈, 사용 예정 시점, 중도 인출 필요 여부, 선호 방식, 가능한 우대조건을 입력받는다.
2. 상품 데이터 테이블: 네이버페이 예금/적금/파킹통장 정보를 기준으로 금융사명, 상품명, 유형, 기본금리, 최고금리, 상세 `금리 안내` 원문, 우대조건, 나이/소득 조건, 첫가입 조건, 한도, 가입 기간, 예금자보호 여부, 공식 가입 링크, 업데이트 날짜를 관리한다.
3. 최적 배분 계산: 세후 예상 이자를 기준으로 목돈과 월 저축액을 상품별 한도와 사용자 조건 안에서 배분한다.
4. 결과 화면: 추천 금액, 예상 세전/세후 이자, 일반 입출금통장 대비 추가 이익, 가입 순서, 계산 가정을 보여준다.
5. 알림 신청: 만기, 금리 변동, 재배분 필요 시점을 위한 이메일 또는 카카오 알림 신청을 받는다.

## MVP 제외 사항
- 자동 계좌 개설 또는 실제 금융 거래 실행
- 개인 신용정보, 계좌 잔액, 인증서 기반 데이터 연동
- 복잡한 투자상품 추천, 주식/펀드/코인 포트폴리오 추천
- 검수 없이 완전 자동 크롤링 결과를 계산에 바로 반영하는 데이터 파이프라인
- 유료 구독 결제와 상세 리포트 판매 기능

## 데이터 업데이트 정책
- 목록 데이터는 정기적으로 확인한다.
- 상세 페이지의 `금리 안내`는 신규 상품, 변경 상품, 캐시에 없는 상품만 다시 수집한다.
- 전날과 동일한 상품은 상세 조건 캐시를 재사용한다.
- 목록에서 사라진 상품은 삭제하지 않고 검수 대상으로 표시한다.

## 디자인
- 매일 쓰는 금융 도구처럼 조용하고 신뢰감 있게 만든다.
- 첫 화면은 계산기와 결과를 중심으로 구성하고 마케팅 랜딩처럼 만들지 않는다.
- 데이터 기준일, 계산 가정, 위험 고지를 작게 숨기지 않는다.
- 색상은 밝은 배경, 진한 본문 텍스트, 절제된 녹색/파란색 포인트를 기본으로 한다.
